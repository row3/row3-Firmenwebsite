Notifications API

Beschreibung

Die Notifications API ist eine Schnittstelle um den End-Nutzer über bestimmte Ereignisse bzw. Vorgänge, die ihn betreffen, zu benachrichtigen. Dafür wird der Nutzer außerhalb des Kontexts einer Webseite alarmiert. Beispiele dafür sind unter anderem der Eingang einer Email oder auch die Erinnerung an einen Termin oder an einen Geburtstag.

Aufgerufen werden die Benachrichtigungen durch den Konstruktor Notification.Notification().

Benachrichtigungen können nur angezeigt werden, wenn der Nutzer seine Erlaubnis dazu gegeben hat. Da gibt es drei Eigenschaften: default (Nichterfüllung), denied (Verweigerung) u nd granted (Genehmigung). Au ßerdem gehört der Titel und der Text der Notification, sowie die Sprache dazu. Aber auch der Körper, die ID und das URL des Bildes, welches als Symbol verwe ndet werden kann, können in einer Notification eingefügt werden. Des Weiteren kann eine strukturierte Ko pie der Daten wiedergegeben werden und es kann spezif iz iert werden, ob die Benachrichtigung leise, daher ohne Klang oder Vibration widergegeben werden soll.

Mit den Methoden der Notifications API kann die die Erlaubnis des Nut zers eingeholt werden, die Benachrichtigung anzuzeigen. Es kann außerdem eingestellt werden, dass das Prog ramm ein e Benachrichtigung automatisch schließt.

Der Ablauf einer Notification kann in drei Schritte unterteilt werden. Im ersten Schritt wird überprüft, ob der Browser die Notifications unterstützt. Wenn das nicht der Fall ist, gibt es eine Warnung. Im Schritt zw ei wird überprüft ob es eine gewährleistete Erlaubnis für Notifications gibt . Wenn nicht, wird der Nutzer um Erlaubnis gefragt. Sobald eine Erlaubnis vorhanden ist, kann im letzten Schritt, die Notification erstellt werden.

Quelltext

<script>
function notifyMe() {
    // Let's check if the browser supports notifications 
    if (!("Notification" in window)) {
        alert("This browser does not support desktop notification");
    }

    // Let's check whether notification permissions have already been granted 
    else if (Notification.permission === "granted") {
        // If it's okay let's create a notification 
        var notification = new Notification("Hi there!");
    }

    // Otherwise, we need to ask the user for permission 
    else if (Notification.permission !== 'denied') {
        Notification.requestPermission(function (permission) {
            // If the user accepts, let's create a notification 
            if (permission === "granted") {
                var notification = new Notification("Hi there!");
            }
        });
    }

    // At last, if the user has denied notifications, and you  
    // want to be respectful there is no need to bother them any more. 
}
</script>

File Api

Beschreibung

Mit der File API gibt es eine Möglichkeit auf lokale Dateien zuzugreifen und diese in einem HTML-Dokument darzustellen. Beispielsweise lassen sich so Dateien vor einem Upload im Browserfenster anzeigen und überprüfen. Die File-API kann nur auf Dateien zugreifen die z.B. über ein Formular ausgewählt werden.

Schreibgeschützte Informationen wie z.B. Name, Dateigröße, Mimetyp und Verweis auf den Dateihandle können bereitgestellt werden ohne, dass das Dokument hochgeladen werden muss.

Bei der Benutzung der File-API können mehrere Methoden zum Laden einer Datei genutzt werde wie z.B. die Auswahl über Drag and Drop oder die Verwendung der Formulareingabe für die Auswahl. Die einfachste Methode dabei ist die Verwendung eines Standardelements vom Typ input type="file" (Formulareingabe). JavaScript gibt die Liste der ausgewählten File-Objekte als FileList zurück.

Den Quelltext für die Verwendung der Formulareingabe könnt ihr hier nachlesen.

Quelltext 

<input type="file" id="files" name="files[]" multiple />
<output id="list"></output>

<script>
function handleFileSelect(evt) 
 {
      var files = evt.target.files; // FileList object

       // files is a FileList of File objects. List some properties.
       var output = [];
       for (var i = 0, f; f = files[i]; i++)  
       {
       output.push('<li><strong>', escape(f.name),   '</strong> (', f.type || 'n/a', ') - ',
       f.size, ' bytes, last modified: ',
       f.lastModifiedDate.toLocaleDateString(), '</li>');
        }
     document.getElementById('list').innerHTML = '<ul>' + output.join('') + '</ul>';
 }

  document.getElementById('files').addEventListener('change'
handleFileSelect,false);
</script>
