Canvas - Die HTML Leinwand

Canvas ist eine Möglichkeit dirket in HTML5 Formen zu zeichnen. Dazu wird eine Fläche (das Canvas) mit angegebene Höhen- und Breitenangaben erstellt.
In dieser Fläche werden dann per Java Script die Formen zu zeichnen. Dabei ist so gut wie alles möglich: Kreise, Rechtecke, Schriften, Bilder, Farbverläufe, Pfade, ...

Hier ist ein kleines Beispiel dafür wie man mit Canvas ein Kreis und ein Rechteck zeichnen kann:

Quelltext

<canvas id="CanvasID" width="CanvasHöheInPixel" height="CanvasBreiteInPixel">
</canvas>

<script>
 
        var e=document.getElementById("meinKreisCanvas");
        var canv=e.getContext("2d");
        //Kreis
        canv.fillStyle= "darkblue";
        canv.arc(150,250,150,0,2*Math.PI);
        canv.fill();
        //Rechteck
        canv.fillStyle = "darkblue";
        canv.fillRect(75, 0, 150, 150);
 
</script>
